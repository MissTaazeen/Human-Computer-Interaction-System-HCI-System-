Hand Gesture HCI System (Phase 3 Prototype)

How to Run:
1. Double-click HandGesture.exe
2. Allow camera permissions
3. Use gestures:

Gestures:
- Move Cursor: Index fingertip movement
- Click: Short pinch (thumb + index)
- Drag: Hold pinch for longer duration

Controls:
- Press 'Q' to quit

System Requirements:
- Windows 10/11
- Webcam required

Known Issues:
- Drag works better on file explorer than browser scrollbars
- Gesture thresholds may vary depending on lighting
